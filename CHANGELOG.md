## [v0.1.0] - 2026-03-27

### Overview

Initial public release of Ascend Skills plugin for Claude Code. This plugin provides a comprehensive set of skills for Huawei Ascend NPU development, including model deployment, environment configuration, and performance optimization.

### Added

#### Skills (9 total)

**ascend-deploy** - Model Deployment
- `aisbench-perf-acc` - vLLM model accuracy and performance benchmarking
- `evalscope-perf-acc` - Comprehensive LLM evaluation framework
- `modelscope-cli` - ModelScope model and dataset download tool
- `vllm-ascend-deploy` - vLLM deployment on Ascend NPU servers

**ascend-env** - Environment Configuration
- `docker-env` - Ascend NPU environment in Docker containers
- `hccn-tools` - HCCN network configuration and diagnostics
- `hccl-bench` - HCCL cluster communication benchmarking
- `mirror-proxy` - Mirror source and proxy configuration

**ascend-optim** - Performance Optimization
- `vllm-ascend-prof` - vLLM inference performance analysis and monitoring

### Features

- Automatic skill triggering based on user context
- Comprehensive scripts and configuration templates
- Reference documentation with version and timeliness markers
- URL.md files for each skill with curated reference links
