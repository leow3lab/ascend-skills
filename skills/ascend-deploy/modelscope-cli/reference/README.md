# ModelScope CLI 使用指南

## 模型参数量计算

### 参数量计算原理

模型参数量可以通过模型权重文件大小和数据精度来推算：

```
参数量(B) = 模型文件总大小(GB) / 每参数字节数
```

### 常见精度类型

| 精度类型 | 每参数字节数 | 文件名标识 | 说明 |
|-----------|---------------|------------|------|
| FP32 | 4.0 | `*FP32*` | 32位浮点，全精度 |
| BF16/FP16 | 2.0 | `*BF16*`, `*FP16*` | 16位浮点，半精度 |
| W8A8Z/W8A8 | 1.0 | `*W8A8Z*`, `*W8A8*` | 8位量化 |
| FP8/INT8 | 1.0 | `*FP8*`, `*INT8*`, `*Q8*` | 8位浮点/整数 |
| W4A8/Q4 | 0.5 | `*W4A8*`, `*Q4*` | 4位量化 |

### 使用示例

```bash
# 统计单个模型的参数量
bash scripts/run_report_param.sh ./Eco-Tech/Qwen3.5-397B-A17B-w8a8-mtp

# 输出示例：
# ========================================
# 模型: Qwen3.5-397B-A17B-w8a8-mtp
# ========================================
# 权重文件数量: 4
# 模型总大小: 397.00 GB
# 数据精度: W8A8Z/W8A8 (每参数 1 字节)
# 推测参数量: 397.00 B (>70B)
```

### 参数量范围参考

| 参数量范围 | 典型模型示例 |
|-----------|--------------|
| <1B | Tiny, Small 系列 |
| 1-7B | Qwen2-1.8B, Llama-3.2B |
| 7-13B | Qwen2-7B, Llama-3-8B |
| 13-34B | Qwen2-14B, Qwen2-32B |
| 34-70B | Qwen2-72B, Qwen3.5-72B |
| >70B | Qwen3.5-397B-A17B-w8a8-mtp |

### 相关脚本

- `scripts/run_report_param.sh` - 模型参数量统计报告脚本
- 支持批量分析多个模型目录
- 自动识别权重文件和精度类型
- 生成汇总报告便于对比
